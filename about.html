About

Meow-finder is a website for cat lovers that shows different cat breeds with pictures and details. Using quality filtering tools, it makes it easy for users to quickly find and choose the perfect cat for their household.

This website was developed using HTML, CSS, Bootstrap, Javascript, jQuery, Web API calls from NinjaAPI and TheCatAPI.
